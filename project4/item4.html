Rave Night Flyer

This flyer was one of my first Adobe InDesign projects in which my task was to design a flyer for any event. As someone who enjoys electronic dance music, I decided to create an EDM themed flyer. This flyer features Apotek Extended as the main font for all text and the background image which conveys an EDM theme to the audience.
